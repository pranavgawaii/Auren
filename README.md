# Auren — AI Execution Layer

Your inbox, calendar, and GitHub. Executed in one line.

Auren is a unified execution layer designed to bridge the gap between thinking and doing. It collapses multi-step administrative workflows across Gmail, Google Calendar, and GitHub into a single natural language command. Built with a strict **human-in-the-loop** lifecycle (Analyze → Plan → Approve → Execute) to ensure reliability and safety.

---

## ⚡ Core Features

- **Action-Oriented Workflows**: Draft emails, schedule meetings, or open structured GitHub issues from a single command line.
- **Cross-Platform Context**: Automatically reads Git commits to summarize standups, checks Google Calendar availability, and references historical email threads to draft replies.
- **Keyboard-First Interface**: Complete keyboard navigation calibrated for extreme speed and efficiency.
- **Real-Time Synchronization**: Instant email threading, Google Calendar integration, and live agent execution logs.
