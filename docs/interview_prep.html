1. Project #1: Auren — AI Task Execution Engine
⚡ 30-Second Pitch for Auren

"I built Auren to eliminate context-switching between developer tools. Instead of manually juggling Gmail, Google Calendar, and GitHub to coordinate daily engineering tasks, developers issue plain-text commands like 'Schedule a sync with Rahul tomorrow 3 PM and open a GitHub issue for the auth bug'.

Auren parses workspace context using OpenRouter AI, constructs a structured multi-step execution plan, presents a human-in-the-loop review interface for 1-click approval, and executes actions concurrently across APIs via Corsair SDK and MongoDB Atlas."

🎯 2-Minute Architectural Deep-Dive for Auren

"Auren is built with Next.js 16 App Router using Server Actions to eliminate client JS bundle bloat and keep API keys strictly on the server. When a command is submitted, an AI router parses context and outputs a strongly-typed PlannedAction[] schema.

To prevent unintended side effects, we enforce a strict Human-in-the-Loop pattern: proposed actions are saved with a PENDING status in MongoDB Atlas. The user previews exact parameters in an interactive UI. Upon explicit approval, Auren dispatches parallel requests through the Corsair App SDK to Gmail, Google Calendar, and GitHub, marking execution COMPLETED."

2. Project #2: Mnemo — AI Context & User Preference Memory Engine
🧸 Mnemo Explained Like You Are 10 Years Old (Pure Backend!)

Imagine every time you chat with an AI chatbot, it forgets your name, your programming language, and what project you are working on the next day. You have to explain everything over and over again!

Mnemo is a digital long-term memory system for AI Chatbots.

1. Extracting Facts: When you chat, Mnemo's backend API extracts important facts about you (like 'Samiksha codes in Python') and saves them as a JSON profile in a database tied to your user ID.

2. Saving Preferences: If your preferences change (e.g. you say 'I switched to Go'), Mnemo automatically updates the stored JSON document so old facts don't confuse the AI.

3. Fast Memory Injection: Next time you open a new chat, Mnemo fetches your saved user profile in less than 0.03 seconds and automatically injects it into the AI prompt so it remembers everything across sessions!

⚡ 30-Second Technical Pitch for Mnemo

"I engineered Mnemo as a pure backend REST API memory service for AI applications. It automatically extracts user preferences and facts from chat messages, stores them in structured MongoDB documents indexed by user ID, and injects user context into fresh AI prompts to enable seamless cross-session memory without manual re-entry."

🛠️ Pure Backend Tech Stack & Key Features

• Pure REST API Architecture: Built in Node.js & Express (or Python FastAPI) with zero frontend UI requirements. Tested via Postman / Swagger (`/docs`).

• Automated Preference Extraction: Passes unstructured chat logs through OpenRouter / Claude Haiku to parse JSON user profiles.

• Sub-30ms Context Retrieval: MongoDB Atlas indexed B-tree queries fetching user profiles for fast system prompt injection.

• Security Middleware: JWT authentication, input validation, and sliding-window rate limiting.
